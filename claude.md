# Proyecto ZOE

## Contexto

ZOE es una plataforma de gestión documental para el Programa del Bachillerato Internacional (IB) del Colegio Técnico Digital Julio Flórez.

El proyecto utiliza:

* Python
* HTML
* CSS
* JavaScript

No utilizar:

* React
* Next.js
* Angular
* Vue
* Svelte
* Tailwind

Toda mejora debe realizarse utilizando únicamente el stack actual.

---

## Objetivo principal

Convertir ZOE en una aplicación SaaS educativa moderna, profesional y escalable.

Inspiraciones:

* Notion
* Linear
* Stripe Dashboard
* GitHub
* Google Workspace

---

## Archivos prioritarios

Analizar primero:

app/templates/
app/static/styles/

---

## Archivos secundarios

Analizar solamente si son necesarios:

config.py
main.py

---

## Ignorar inicialmente

No dedicar tiempo a:

* conexión a base de datos
* migraciones
* autenticación avanzada
* optimización de consultas
* despliegue
* Docker
* CI/CD

Estas áreas no forman parte del objetivo actual.

---

## Tarea principal

Realizar exclusivamente mejoras visuales y estructurales.

Prioridades:

1. Dashboard
2. Layout
3. Sidebar
4. Navbar
5. Formularios
6. Tablas
7. Reportes
8. Cronograma
9. Mensajería

---

## Restricciones

No eliminar funcionalidades existentes.

No cambiar nombres de rutas.

No cambiar estructura de carpetas.

No modificar lógica de negocio sin justificación.

No crear dependencias innecesarias.

No migrar el proyecto a otro framework.

---

## Flujo obligatorio

Antes de modificar archivos:

1. Analizar proyecto.
2. Generar diagnóstico.
3. Clasificar problemas:

   * Alto impacto
   * Medio impacto
   * Bajo impacto
4. Proponer plan de acción.
5. Esperar aprobación antes de realizar cambios masivos.

---

## Estilo visual esperado

Diseño moderno SaaS.

Características:

* bordes suaves
* sombras discretas
* espaciado consistente
* tipografía profesional
* diseño responsive
* tarjetas modernas
* tablas modernas
* formularios modernos
* accesibilidad
* modo oscuro preparado para implementación futura

## Objetivo final

La plataforma debe parecer un sistema profesional utilizado por instituciones educativas internacionales.
